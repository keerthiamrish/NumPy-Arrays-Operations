# 🔢 NumPy Basics - Array Operations

## 📌 Project Overview
This project demonstrates the fundamentals of NumPy, a powerful Python library used for numerical computing. It covers array creation, mathematical operations, and statistical functions.

---

##  Objectives
- Create 1D, 2D, and 3D arrays
- Perform basic arithmetic operations
- Use NumPy functions for analysis
- Understand array reshaping and generation

---

## Features
- Creation of 1D, 2D, and 3D arrays
- Arithmetic operations (addition, subtraction, multiplication, division)
- Statistical functions:
  - mean()
  - sum()
  - std()
- Array manipulation:
  - reshape()
  - arange()
  - linspace()
- Find indices of maximum and minimum values

---
##  Technologies Used
- Python
- NumPy
